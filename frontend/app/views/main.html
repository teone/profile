Short Bio

Ciao, sono Matteo Scandolo, web enthusiast nato nel maggio '86. Lavoro quotidianamente con un team eterogeneo di esperti che collaborano nella creazione di Rich Application web e mobile oriented.

In particolare le mie conoscenze sono orientate a:
  • Markup Html5 - valido secondo gli standard w3c
  • Advanced Css3 - Anche tramite l'utilizzo di Less
  • UX design - con software come Axure RP e Balsamiq Mockup
  • Costruzione di layout full responsive - principalmente sfruttando framework come Bootstrap o Foundation
  • Javascript - con librerie e framework (jQuery/AngularJs/Intel AppFramework)
  • Applicazioni mobile crossplatform (PhoneGap)
  • Applicazioni Server-Side (Php e mySql con l'utilizzo di framework quali Yii o LaRavel)
  • Utilizzo di API di terze parti - Facebook, Linkedin, Google, Dropbox e altre
  • Versioning con Git
  • Seo - Conoscenza della struttura dei tag dedicati (meta, :og e microformati schema.org), generazione dinamica di sitemap e utilizzo di Google Analitycs e WebmasterTools
  • Buon utilizzo di tutta la Creative Suite Adobe e altri tool (Automator, GIMP, ...)
Sempre interessato alle ultime novità da gennaio lavoro con, e non voglio più fare a meno di:
  • Database non relazionali (MongoDB)
  • Javascript Power Tools (Grunt - Yeoman - Bower)
  • Stack MEAN

Ma la mia vita non finisce qui, mi piace lavorare in gruppi dove si effettui un continuo scambio di know-how per aumentare le potenzialità di tutto il team, mentre nel tempo libero adoro praticare sport all'aria aperta come surf, ciclismo, snowboard e arrampicata.
